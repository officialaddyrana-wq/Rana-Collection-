# Pulse+ Social Media App

Pulse+ is a full-featured social media prototype built using vanilla HTML, CSS, and JavaScript.

## Included social media features

- Custom profile setup (name, avatar, bio).
- Story rail generated from active posters.
- Post creation with text, optional image URL, and hashtags.
- Like, comment, repost, bookmark, share, edit, and delete actions.
- Search across users, content, and hashtags.
- Feed sorting (newest, most liked, most commented).
- Trending hashtag panel.
- Notifications center.
- Activity summary and saved-post mini list.
- Light/Dark theme toggle.
- Local persistence with `localStorage`.

## Run locally

```bash
python3 -m http.server 8080
```

Open: http://localhost:8080
