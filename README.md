# RANA COLLECTION

Enhanced single-page e-commerce website starter for **RANA COLLECTION**.

## Features

- Responsive storefront with search, sorting, and category filtering
- Cart panel with quantity increase/decrease controls and total calculation
- Cart persistence via `localStorage` key: `rana_cart_v3`
- Theme persistence via `localStorage` key: `rana_theme`
- **User Login Portal** (demo: any non-empty username/password)
- **Admin Login Portal** (demo credentials: `admin` / `admin123`)
- **Admin Product Portal** to add products dynamically to the catalog
- Admin-added products persist via `localStorage` key: `rana_admin_products`
- User/admin session persistence via `localStorage` key: `rana_user_session`

## Run

Open `index.html` directly in your browser.
